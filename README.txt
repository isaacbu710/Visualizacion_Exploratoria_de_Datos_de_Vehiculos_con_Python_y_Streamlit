# Proyecto de Análisis de Vehículos Usados

## Descripción del Proyecto

Este proyecto consiste en una aplicación web interactiva desarrollada con **Streamlit** y **Plotly** que permite visualizar y analizar un conjunto de datos sobre vehículos usados. La aplicación permite a los usuarios interactuar con un conjunto de datos sobre vehículos en venta, mostrando diferentes tipos de gráficos para ayudar en el análisis.

## Funcionalidad

La aplicación ofrece dos tipos de gráficos interactivos:

1. **Histograma de Odometer**: Muestra un histograma de la columna "odometer" (kilometraje) de los vehículos en el conjunto de datos, lo que permite observar la distribución de los kilometrajes de los vehículos.
   
2. **Gráfico de dispersión**: Permite a los usuarios ver una relación entre el kilometraje (`odometer`) y el precio (`price`) de los vehículos mediante un gráfico de dispersión.

Los usuarios pueden elegir mostrar cada uno de estos gráficos de manera independiente utilizando **checkboxes**.
